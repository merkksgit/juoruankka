## Overview

The site uses a dark theme with a warm, vibrant color palette centered around blues, yellows, and oranges.

## Color Variables

### Primary Colors

- Primary Color: #233c4b - Deep blue-grey (used for text on bright backgrounds and footer)
- Secondary Color: #fac846 - Warm yellow/gold (used for highlights, buttons, and accents)
- Accent Color: #ff7d2d - Vibrant orange (used for CTAs, active states, and emphasis)
- Success Color: #fac846 - Same as secondary (yellow/gold)

### Background Colors

- Background: #1a2c38 - Dark blue-grey (main background)
- Card Background: #2a4456 - Medium blue-grey (card surfaces)

### Text Colors

- Text Color: #f5f5f5 - Off-white (primary text)
- Text Light: #b8c5cf - Light blue-grey (secondary text, meta information)

### Structural

- Border Color: #3d5a6f - Medium blue-grey (borders and dividers)

### Design Features

- Header: Linear gradient from #fac846 (yellow) to #ff7d2d (orange) - creates a warm, eye-catching banner
- Typography: Uses BigBlueTerm custom font for the logo, Segoe UI Variable for body text
- Shadows: Layered box shadows for depth (0 2px 8px rgba(0,0,0,0.3) and hover states)
- Theme: Professional dark mode with high contrast and warm accent colors for excellent readability
